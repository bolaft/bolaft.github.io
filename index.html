Objectifs du TD :

Développer une application en Python consommant des données "open data" et exposant des services REST les utilisant.


L'objectif final, c'est d'obtenir une interface web (HTML/JS) qui permette de rechercher via des formulaires des équipements, des salles et des installations sportives.


L'application web obtient les données d'un serveur exposant des services REST. La base de données du serveur est alimentée par l'open data des Pays de la Loire.

Vous devez développer :

  • Le module du serveur qui importe les données en base
  • Le module du serveur qui expose les services REST
  • Un module d'administration pour modifier ou supprimer les donnnées (optionnel)
  • L'application web qui permet de faire les recherches

Plan du TD :

  1. Prise de contact avec Python
  2. Accès à une base de données en Python
  3. Accès à des données externes : fichiers et de services Web
  4. Client-serveur : création d'un serveur
  5. Exposition de services Web
  6. Utilisation des données géographiques (optionnel)
  7. Développement d'une interface complète

def compute(a, b, double=False):
    x = a + b
    if double:
        x = x * 2
    return x

compute(2, 3) # 5
compute(7, 2, double=True) # 18

def generate_doubles(l):
    for x in l:
        yield x * 2

for d in generate_doubles([1, 5, 3]):
    print(d) # 2, 10, 6

    Arguments et options

    
from optparse import OptionParser


def do_actual_stuff(args, opts):
    print("The input file is {0}".format(args[0]))
    if opts.quick:
        print("Fast!")


def parse_args():
    op = OptionParser(usage="usage: %prog [opts] input_file")

    op.add_option("-q", "--quick",
        dest="quick",
        default=False,
        action="store_true",
        help="if true, makes things go faster")

    return op.parse_args()


if __name__ == "__main__":
    options, arguments = parse_args()

    do_actual_stuff(arguments, options)

    Exemple de serveur :

    
from bottle import route, template, run


@route('/hello/')
def index(name):
    return template('Hello {{var}}!', var=name)

run(host='localhost', port=8080)
    1. Exécuter le script
    2. S'assurer que le port 8080 est disponible :
      • sudo service apache2 stop
      • sudo service tomcat6 stop
    3. Réquêter le serveur :
      • http://localhost:8080/hello/world
      • curl -XGET http://localhost:8080/hello/world
